HTML Form Elements

Tag Description
<form> The wrapper element for all HTML forms.
<input> The field in which the user inputs data.
<label> This is a complementary element for the <input> tag
<select> This element serves as a dropdown menu, containing multiple options.
<option> This represent an item within a dropdown menu.
<textarea> This is an input field which allows users to enter multi-line text and allows box resizing.
<fieldset> This HTML element is used to group several controls in a form.
<legend> This element serves as a caption for the fieldset.

Form Elements Attributes

Attributes Description
for=" " Used for Label elements to indicate which input/form element does the label discribe.
id=" " Used to give an element a specific identification which can be used as the for attribute value.
name=" " This is a necessary form attribute that describe that form element.
required This attribute is given to a form input/dropdown/textarea that prevents the form from submtting until those fields are filled.
placeholder Used in input elements that has a default text in the entry field that gives a description or example of info type.
type=" " This is used in input fields which state whether it's a text, email, number, password, etc.
